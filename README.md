## 📖 Overview
This product is a command-line interface (CLI) tool that scans local scripts to auto-generate a 'Tool Manifest' (JSON schema) for instant AI agent integration. It solves the problem of manually extracting functional value from codebases, which is often ignored by traditional code analysis tools. The tool uses Python's built-in 'ast' library to parse Abstract Syntax Trees (AST) and extract public function definitions, type hints, and docstrings, compiling them into a standardized 'toolbox.json' manifest. This tool is for developers and businesses looking to integrate their codebases with AI agents, providing a free and MIT-licensed solution that requires no external dependencies or complex configuration.
